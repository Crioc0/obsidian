# Заметка
Эффективно работает при сортировке небольшого количества элементов.
Псевдокод сортировки
```
for j ← 2 to length[A]
	do key ← A[j]
		// Вставка элемента A[j] в отсортированную
		// последовательность A[1..j − 1]
		i ← j - 1
		while i > 0 и A[i] > key
			do A[i + 1] ← A[i]
				

```
