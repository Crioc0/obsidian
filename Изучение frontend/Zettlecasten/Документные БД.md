# Заметка
Вместо таблиц в этих БД используют коллекции — нежёсткие структуры данных, которые состоят из документов, или записей.
Пример запесей в документной БД
```json
{
    "id": "1234252",
    "name": "Внук Виталия Сергеевича",
    "madeOrder": true
},
{
    "id": "1208772",
    "name": "Стас Басов",
}
```
