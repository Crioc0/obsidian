# Заметка
JavaScript не имеет синтаксиса для объявления свойств классов, но в TypeScript такой синтаксис присутствует. 

Верися для typescript
```ts
class Person {
	firstName: string;
	lastName: string;
	age: number
}

const p = new Person()
p.firstName = "John"
p.lastName = "Smith"
p.age = 25
```
