Псевдокод - код, удобный для человеческого восприятия, но непонятный для машины.

```
function maximum(A, B, C)
    if A > B
        if A > C
            max ← A
        else
            max ← C
    else
        if B > C
            max ← B
        else
            max ← C
    print max
```

Такой код полностью игнорирует синтаксические правила языков программирования
