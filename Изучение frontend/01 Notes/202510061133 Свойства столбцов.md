Свойства столбцов отвечают за более простые вещи, например за длину поля, его значение по умолчанию и так далее. Прописываются они внутри декоратора `@Column` следующим образом:
```ts
@Column({
  type: "varchar",
  length: 150,
  unique: true,
  // ...
})
name: string;
```
