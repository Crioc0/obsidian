**Назначение:** Создание множества похожих объектов (пользователи, элементы меню и т.д.)

**Синтаксис:**

js

function User(name) {
    this.name = name;
    this.isAdmin = false;
}
let user = new User("Jack");

**Соглашения:**

1. Имя начинается с большой буквы
    
2. Вызывается только через оператор `new`
    

**Что происходит при вызове с `new`:**

1. Создаётся новый пустой объект, присваивается `this`
    
2. Выполняется тело функции (модифицирует `this`, добавляет свойства)
    
3. Возвращается значение `this`
    

**Немедленно вызываемый конструктор:**

js

let user = new function() {
    this.name = "John";
    // сложная логика создания
};

- Создаётся и сразу вызывается
    
- Не может быть повторно использован
    
- Цель: инкапсуляция кода создания объекта
    

**Возврат значений:**

- Если `return` вызывается с объектом → возвращается объект вместо `this`
    
- Если `return` вызывается с примитивом → примитив игнорируется
    

**Создание методов:**

js

function User(name) {
    this.name = name;
    this.sayHi = function() {
        alert("Привет, " + this.name);
    };
}

Конструкторы могут иметь параметры для гибкого создания объектов с разными свойствами и методами.
